# 🏠 DreamHome

> *Find Your Perfect Dream Home*

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/)
[![Flask](https://img.shields.io/badge/Flask-2.0%2B-green.svg)](https://flask.palletsprojects.com/)
[![SQLite](https://img.shields.io/badge/SQLite-3.x-orange.svg)](https://www.sqlite.org/)

**DreamHome** is a Python Flask web application for discovering, exploring, and booking architectural dream homes.

---

## ✨ Features

| Module | Features |
|--------|----------|
| 👤 **User** | Register, Login, Browse Projects, Book Now, Dashboard, Reviews |
| 👑 **Admin** | Manage Projects, Users, Bookings, Reviews |
| 🎨 **Other** | Responsive Design, Profile Upload, Secure Auth |

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Python Flask |
| Database | SQLite |
| Frontend | HTML, CSS, JavaScript |
| Templating | Jinja2 |
