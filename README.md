## 📌 Overview

**DreamHome** is a comprehensive web-based platform built with **Python Flask** that bridges the gap between dream home seekers and architectural project providers. The system streamlines the process of discovering, exploring, and booking residential projects through an intuitive user interface.

### 🎯 Purpose
- Simplify home discovery and booking process
- Provide detailed architectural project information
- Enable seamless consultation scheduling
- Build trust through user reviews and ratings

---

## ✨ Features

### 👤 User Module

| Feature | Description |
|---------|-------------|
| 🔐 **Authentication** | Secure registration & login system |
| 👤 **Profile Management** | Update personal info & profile picture |
| 🏠 **Project Browsing** | Explore various architectural projects |
| 📖 **Project Details** | View complete project specifications |
| 📅 **Book Now** | Schedule consultations instantly |
| 📊 **Dashboard** | Track all bookings & activities |
| ⭐ **Reviews** | Share feedback & read experiences |
| 📞 **Contact** | Direct communication with team |

### 👑 Admin Module

| Feature | Description |
|---------|-------------|
| 📋 **Project Management** | Add, edit, delete projects |
| 👥 **User Management** | View and manage users |
| 📝 **Booking Management** | Handle consultation requests |
| 💬 **Review Moderation** | Manage user feedback |

### 🎨 Additional Features

- Responsive Design for all devices
- Image upload for profiles
- Real-time form validation
- Secure password hashing
- Session management
